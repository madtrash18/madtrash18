# so_long

> 42 Gyeongsan — Circle 2

MiniLibX 라이브러리를 사용해 구현한 2D 탑뷰 미니 게임입니다.  
플레이어는 맵을 탐색하며 모든 아이템을 수집하고 출구로 이동해야 합니다.

---

## 게임 규칙

- **목표:** 맵에 있는 모든 아이템(`C`)을 수집한 뒤 출구(`E`)로 이동
- **이동:** `W`/`A`/`S`/`D` 또는 방향키
- **이동 횟수**는 터미널에 실시간으로 출력
- `ESC` 또는 창 닫기 버튼으로 종료

---

## 맵 형식 (`.ber`)

```
1111111111111
10010000000C1
1000011111001
1P0011E000001
1111111111111
```

| 기호 | 의미 |
|------|------|
| `1` | 벽 |
| `0` | 빈 공간 |
| `P` | 플레이어 시작 위치 (1개) |
| `E` | 출구 (1개) |
| `C` | 수집 아이템 (1개 이상) |

---

## 맵 유효성 검사

- 직사각형 맵, 벽으로 완전히 둘러싸임
- P, E, C 각각 최소 1개 이상
- BFS/DFS로 경로 가능성(P→모든 C→E) 검증

---

## 구현 내용

- MiniLibX 이벤트 루프 및 키 입력 처리
- XPM 텍스처 로딩 및 타일 기반 렌더링
- 플레이어 이동 및 맵 상태 업데이트
- 맵 파일 파싱 및 전방위 유효성 검사

---

## 실행 방법

```bash
make
./so_long maps/example.ber
```
